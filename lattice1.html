Solving lattice models — Part 1

Today we are solving some square lattice models numerically. From the notes we know that the simplest case of a square lattice with chemical potential \(\mu\) and d-dimension is 

\begin{align}
  H = -\frac{t}{2}\sum_{i}^{N} c_{\sigma,i+1}^{\dagger} c_{\sigma,i} + \mbox{h.c.}
\end{align}
\begin{align} H = -\frac{t}{2}\sum_{i}^{N} c_{\sigma,i+1}^{\dagger} c_{\sigma,i} + \mbox{h.c.} \end{align} \begin{align} H = -\frac{t}{2}\sum_{i}^{N} c_{\sigma,i+1}^{\dagger} c_{\sigma,i} + \mbox{h.c.} \end{align}

where for the closed system we can Fourier Transform1 to 

\begin{align}
  H = \sum_{k} \left( -t \cos k a\right) c_{\sigma,k}^{\dagger} c_{\sigma,k}
\end{align}
\begin{align} H = \sum_{k} \left( -t \cos k a\right) c_{\sigma,k}^{\dagger} c_{\sigma,k} \end{align}

We can just ignore the \(\sigma\) for now. In this form the Hamiltonian is called ´diagonalized´. The eigenvalues are simply the cosine function and can be easily plotted in the first Brillouin zone2. 

k,t,a = var('k','t','a')
t = a = 1
plot(-t * cos(k*a), \
     (k, -pi/a, pi/a), \
     axes_labels=["$k$", "$E_k$"], \
     figsize=5)

We could add the chemical potential to this simple model 

\begin{align}
  H =-\frac{t}{2}\sum_{i}^{N} \left(c_{\sigma,i+1}^{\dagger} c_{\sigma,i} 
  + \mbox{h.c.} \right) - \mu \sum_{i}^{N}c_{\sigma,i}^{\dagger} c_{\sigma,i}
\end{align}
\begin{align} H =-\frac{t}{2}\sum_{i}^{N} \left(c_{\sigma,i+1}^{\dagger} c_{\sigma,i} + \mbox{h.c.} \right) - \mu \sum_{i}^{N}c_{\sigma,i}^{\dagger} c_{\sigma,i} \end{align}

the result would be just a shift in the energy. 

\begin{align}
  H = \sum_{k} \left(\mu-t \cos k a\right) c_{\sigma,k}^{\dagger} c_{\sigma,k}
\end{align}
\begin{align} H = \sum_{k} \left(\mu-t \cos k a\right) c_{\sigma,k}^{\dagger} c_{\sigma,k} \end{align}

resulting in 

k,t,a,mu = var('k','t','a','mu')
mu = t = a = 1
plot(mu -t * cos(k*a), \
     (k, -pi/a, pi/a), \
     axes_labels=["$k$", "$E_k$"], \
     figsize=5)

So far, so good. Note that for D-dimensional square lattice with chemical potential \(\mu\) we can easily expand this expression as 

\begin{align}
  H_D = \sum_{\vec{k}} \left(\mu-t \sum_{j}^{D}\cos k_j a\right) c_{\sigma,\vec{k}}^{\dagger} c_{\sigma,\vec{k}}
\end{align}
\begin{align} H_D = \sum_{\vec{k}} \left(\mu-t \sum_{j}^{D}\cos k_j a\right) c_{\sigma,\vec{k}}^{\dagger} c_{\sigma,\vec{k}} \end{align}

Where \(k_j\) is the jth element of the quasi-momentum vector \(\vec{k} = \left( k_1, k_2, \ldots, k_D\right)\).

We could then try to solve this model with an open system, that is, we cannot use the Fourier Transform to ´diagonalize´ the Hamiltonian anymore, instead we solve it directly by ´hand´. Example for \(n=10\) atoms in the linear chain. 

import numpy as np
import matplotlib.pyplot as plt

t = 1.0

# First with mu = 0
mu = 0.0
n = 10
H = np.zeros((n,n), float) 
for i in range(n-1):
  H[i,i] = -mu
  H[i,i+1] = -0.5*t
  H[i+1,i] = -0.5*t

eigvals, eigvecs = np.linalg.eig(H)

plt.xlabel('n')
plt.ylabel('$|\psi_n|^2$')

_ = [plt.plot([_ for _ in range(n)], abs(eigvecs.T[j])^2) for j in range(n)]
plt.savefig("./tmp/tmp1.png")
plt.clf()

# Now for mu = 1.0
mu = 1.0
H = np.zeros((n,n), float) 
for i in range(n-1):
  H[i,i] = -mu
  H[i,i+1] = -0.5*t
  H[i+1,i] = -0.5*t
eigvals, eigvecs = np.linalg.eig(H)

plt.xlabel('n')
plt.ylabel('$|\psi_n|^2$')

_ = [plt.plot([_ for _ in range(n)], abs(eigvecs.T[j])^2) for j in range(n)]
plt.savefig("./tmp/tmp2.png")
plt.clf()
